Changelog

Compatibility

  • Added Compatibility with SugarCRM 6.5.0

Changed Behaviors

  • Message "Extension Manager Disconnected" no more displayed to non-admin users

Major Bug Fixes

  • Fixed Bug: an empty PHP file might be erroneously created when uninstalling certain dispage extensions
  • Fixed Bug with MSSQL Database: loading installed extensions in DEM administration panel might fail
  • Fixed Bug with MSSQL Database: error displaying date fields in DEM administration panel
  • Fixed Bug: installing a dispage extension with SugarCRM ver. < 6.0.2 might return errors even if the extension was successfully installed

Minor Bug Fixes

  • Fixed Visualization Problem: the tabs of DEM administration panel might be wrongly visualized while loading the page
  • Fixed Visualization Problem: A small strip might be erroneously displayed at the bottom left of DEM administration panel
  • Fixed visualization bug: auto-adjustment of width of DEM administration panel might fail with several custom Themes
  • Fixed various notices in PHP error log
